# spartaTown
### Unity 게임개발 입문 개인과제
#### Unity 2022.3.2f 버전으로 구현
---
##### 구현기능
1. 캐릭터 만들기
2. 캐릭터 이동
3. 방 만들기
4. 카메라 따라가기(Cinemachine 활용)
5. 캐릭터 애니메이션 추가
6. 이름 입력 시스템
7. 캐릭터 선택 시스템
8. 참석 인원 UI
9. 인게임 캐릭터 선택
10. 인게임 이름 바꾸기
11. 시간 표시
